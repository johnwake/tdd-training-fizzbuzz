# tdd-training-fizzbuzz

A Training Repo to help people practise TDD and BDD.

How to set-up: 

1. Down the Repo from GitHub: 
` git clone git@github.com:johnwake/tdd-training-fizzbuzz.git `

2. Install the dependencies. 
` mvn install `

3. Run the tests: 
` mvn test `
